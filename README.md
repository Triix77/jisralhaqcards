# 🏛 جسر الحق — موقع تصاميم المناسبات

## هيكل المشروع

```
jisr-alhaq-cards/
├── index.html        ← الموقع الرئيسي للزوار
├── admin.html        ← لوحة إعداد التصاميم (لك أنت فقط)
├── designs.json      ← قائمة كل التصاميم
└── images/           ← مجلد صور التصاميم
    ├── ramadan-1.jpg
    ├── fitr-1.jpg
    └── ...
```

---

## خطوات الرفع على GitHub Pages

### 1. أنشئ مستودع جديد
- ادخل على github.com
- اضغط New repository
- اسمه مثلاً: `occasion-cards`
- اتركه Public

### 2. ارفع الملفات
- ارفع: `index.html` و `admin.html` و `designs.json`
- أنشئ مجلد `images/` وارفع فيه صورك

### 3. فعّل GitHub Pages
- Settings → Pages → Source → main → / (root)
- رابط موقعك: `https://USERNAME.github.io/occasion-cards/`

---

## كيفية إضافة تصميم جديد

### الخطوة 1: افتح `admin.html` محلياً
- افتح الملف في المتصفح مباشرة (لا تحتاج سيرفر)

### الخطوة 2: ارفع الصورة وحدد المواقع
- ارفع صورة التصميم
- اسحب النقطة الذهبية لتحديد موقع الاسم
- اسحب النقطة الثانية لتحديد موقع المسمى الوظيفي

### الخطوة 3: انسخ الكود
- اضغط زر "نسخ" تحت الكود JSON

### الخطوة 4: حدّث `designs.json`
- افتح `designs.json`
- أضف الكود الجديد داخل مصفوفة `"designs": [...]`

**مثال:**
```json
{
  "designs": [
    { ... التصميم القديم ... },
    { ... الكود الجديد الذي نسخته ... }
  ]
}
```

### الخطوة 5: ارفع الصورة على GitHub
- ارفع صورة التصميم في مجلد `images/`
- ارفع `designs.json` المحدّث

---

## تنسيق الصور المقترح

- الأبعاد: 1600 × 1200 بكسل
- الصيغة: JPG أو PNG
- الحجم: أقل من 1MB

---

## إضافة مناسبة جديدة

في ملف `designs.json`، أضف في قسم `"occasions"`:

```json
{
  "id": "new-year",
  "label": "العام الجديد",
  "emoji": "🎆"
}
```

يمكنك الاحتفاظ به على جهازك للاستخدام المحلي.
